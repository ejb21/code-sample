Link #1

Cross-site scripting is one of the primary attack vectors on the web. "XSS" occurs when malicious JavaScript is injected into an otherwise trustworthy website. There are numerous ways in which such an insertion may occur, and a number of best practices one should follow to avoid falling prey.

Link #2

SQL injection is THE #1 vulnerability in software and web. Any time user input is written to a database, that input must be "sanitized" to be sure that malicious SQL statements are not accidentally processed by the database management system.

Link #3

"Phishing" is a very broad term referring to all manner of exploits occuring from person to person. Phishing is a type of social engineering whereby a cunning attacker tries to fool or convince an unwitting victim into voluntarily compromising the system — such as by giving away a password.
